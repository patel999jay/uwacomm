## Quick Start

For now, please:

1. Fork the repository at [github.com/patel999jay/uwacomm](https://github.com/patel999jay/uwacomm)
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Development

```bash
# Clone the repository
git clone https://github.com/patel999jay/uwacomm.git
cd uwacomm

# Install in development mode
pip install -e ".[dev]"

# Run tests
pytest tests/ -v

# Run tests with coverage
pytest tests/ -v --cov=uwacomm
```
